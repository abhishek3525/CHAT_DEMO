# Chat Demo

A full-stack chat application with real-time messaging capabilities.

## Project Structure

- **Backend**: Node.js server with authentication and real-time messaging
- **Frontend**: React application with Vite

## Getting Started

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## Features

- User authentication
- Real-time messaging
- Chat history
- Contact list

## Technologies Used

- Backend: Node.js, Express
- Frontend: React, Vite
- Real-time: Socket.IO
